Product Variant Routes

Description

This module allows users to select different routes for the product variants separately/independently from their product template.

Functionality

After installing the Product Variant Routes application , the feature will be enabled.

Go to Inventory - - > Products - - > Products.

Creating a new product with two variants.

When creating a new product, this module by default sets Routes = Buy  for Product Variant.

Enabling additional route “Manufacture” in the newly created product.

We can see the change in the product’s route does not affect the product variants where we can choose different routes for each variant specifically.
